O que é html?

html e uma liguagem de marcação usada para criar paginas e site. HTML permite a criação de documentos interativos, incorporando imagens, links, formulários, e outros elementos multimídia.

Em termos simples, o HTML fornece uma estrutura básica para o conteúdo de uma página web, indicando como o texto deve ser organizado, quais são os títulos, parágrafos, listas, links, imagens, etc. Ele utiliza tags (etiquetas) para definir diferentes elementos no documento, e os navegadores da web interpretam essas tags para exibir o conteúdo da página da forma desejada.

o que é css?

css e uma liguagem de estilização usada para deixar as paginas e site bonitos, deixado a parte visual bonita e atrativa para o publico. o CSS é responsável pela aparência e formatação da página.

O principal objetivo do CSS é permitir que os desenvolvedores controlem o layout, cores, fontes e outros aspectos visuais de suas páginas web de maneira eficiente e consistente. Ele separa a estrutura do documento da sua apresentação, proporcionando maior flexibilidade e facilitando a manutenção.

o que é javascript?

javascript e uma liguagem de programação de alto nível, interpretada e orientada a objetos. usada para manipular o html e o css deixado o site interativos com os usuarios. JavaScript permite que os desenvolvedores criem páginas web interativas, dinâmicas e responsivas.

A principal função do JavaScript é fornecer funcionalidades adicionais às páginas web que não podem ser alcançadas apenas com HTML e CSS. Ele permite a manipulação do conteúdo da página, interação com o usuário, controle de eventos, comunicação com servidores e muito mais.
